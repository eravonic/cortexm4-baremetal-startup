Simple baremetal startup code for an Arm Cortex-M4 processor, where the interrupt vector table is configured specifically for the STM32L476RG microcontroller.

Directory "ozone" contains Segger Ozone project settings if that is the debugger of choice. If using an ST Nucleo-64 development board, Segger provides a ST-Link re-flash utility so that Ozone can be used for free with the board.
